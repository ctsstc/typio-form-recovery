v3.0 Updated!

Typio 3.0 is here, and a lot has changed under the hood.

In this version of Typio I have rewritten a big part of the extension to clean it up and make it easier to continue building upon. Due to the large amount of changes, the risk of something breaking is higher than usual. I have of course gone through extensive testing myself, but with your help I hope to find and iron out the last bugs that needs fixing.

If you notice anything out of the ordinary, please reach out!

Please refresh any currently open web pages in order for Typio to work on them.

So what's changed?

  • New: The entire extension has been refactored to es6 modules
  • New: The options page has been rewritten from ground up
  • New: New and improved build process using webpack
  • Fix: Radio buttons are now saved and restored more reliably
  • Fix: Focus is now restored to the input after using the Quick Restore popup to restore
  • Fix: Lots of other bug fixes and preparation for future releases

Next release coming soon.
