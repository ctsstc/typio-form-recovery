Report an issue

If you encounter any issues with Typio, please let us know. In order for us to fix the issue it is important that you provide as much details as possible in your report. We need to be able to reproduce the issue to find the cause of it. Please include the following in your report:

  • What steps did you take to trigger the problem?
  • Direct link to a page where the problem can be reproduced.
  • If it is a visual bug, screenshots are appreciated.
  • If you are not using Google Chrome, tell us which browser and which version.

Submit an issue report. Remember to be nice.
