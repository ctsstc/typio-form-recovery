Report a bug

If you encounter any problems with Typio, please let us know. In order for us to fix the problem it is important that you provide as much details as possible in your bug report. We need to be able to reproduce the problem to find the cause of it. Please include the following in your bug report:

  • What steps did you take to trigger the bug? How can we reproduce it?
  • Is the bug site specific? Where can we reproduce it?
  • Are you using Google Chrome or another browser? Which version?
  • If it is a visual bug, screenshots are appreciated.

Remember to be nice.
