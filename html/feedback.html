Report an issue

If you encounter any issues with Typio, please let me know. In order for me to fix the issue it is important that you provide as much details as possible in your report. I need to be able to reproduce the issue to find the cause of it. Please include the following in your report:

  • What steps did you take to trigger the problem?
  • Direct link to a page where the problem can be reproduced.
  • If it is a visual bug, screenshots are appreciated.
  • If you are not using Google Chrome, tell me which browser and which version.

Open a ticket. Remember to be nice.

Anything else?
