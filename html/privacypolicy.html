Privacy policy

The main purpose of Typio is to backup and restore form input data. In order for Typio to do this it needs to save your form input as you enter it. Any user input Typio stores is stored only for the purpose of restoration and is never uploaded to any servers. Your data stays on your computer.

You are in control of which sites Typio is enabled on and how long your data is stored. This can be configured in the extension options page.

The source code for Typio is available to view on Bitbucket.
