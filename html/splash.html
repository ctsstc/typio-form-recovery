v2.3 Updated!

Typio 2.3.1 is here!

  • New: Blacklist now compares against full URL instead of hostname (#73). If using regex for blacklisting, you might need to look it over.
  • New: Permissions description page.
  • Change: Save Indicator is now disabled by default (can be re-enabled in options).
  • Change: Removed "Downloads" permission. It was accidentally included ahead of time and was not being used. (#75)
  • Bug fixes and preparation for future release.

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or want to leave feedback, please submit a report in the bug tracker or tweet me at @TypioRecovery.
