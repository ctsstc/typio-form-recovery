v2.0 Updated!

Typio 2.0 is here! This update brings lots of requested features and bug fixes, and under the hood the storage method has been changed which will make storage more reliable. Have a look and see what else is new!

  • New: Regex support for blacklist (#26, #43)
  • New: Option to reset all fields between restorations (#20, #47)
  • New: Option to clone entries upon restoration (#20, #47)
  • New: Support for multiple concurrent same-domain tabs (previously did not work properly due to storage method restrictions) (#21)
  • New: Changed storage method, now much more reliable
  • New: UI refresh
  • Fix: Error messages sometimes thrown by save indicator (#60)
  • Fix: Icon support for sites with strict CSP
  • Fix: Typio now detects subtree changes in contenteditable fields (such as using modifier buttons in rich text fields) (#34)
  • Fix: ...and lots more!

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or want to leave feedback, please submit a report in the bug tracker or tweet me at @TypioRecovery.
