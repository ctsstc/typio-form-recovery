v1.8 Updated!

This update of Typio Form Recovery includes a much requested feature; a restore icon next to inputs for quick access. You can configure it to appear on input focus or double-click, or disable it completely. The restore icon will open the quick restore popup which has been updated to include keyboard support and more options! See a full change list below.

  • New: Restore icon next to inputs (configurable in options)
  • New: Quick Restore now has keyboard support
  • New: Added option to hide Typio in context menu (in favor of keyboard shortcuts)
  • New: Smarter restoration of entries that contain formatting
  • New: "You have disabled Typio on this site" message now gives you the option to enable Typio
  • Fix: Keyboard shortcuts can now be pressed in any order
  • Fix: Fix some positioning issues with Quick Restore popup
  • Fix: "Delete all" now actually deletes all entries, including current session.

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or want to leave feedback, please submit a ticket in the bug tracker or tweet me at @TypioRecovery.
