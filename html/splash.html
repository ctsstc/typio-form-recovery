v1.6.3 Updated!

The last keyboard shortcut update introduced issues on Mac OS so this update reverses the changes for Mac OS and adds the ability to set your own keyboard shortcuts.

  • New: You can now set your own keyboard shortcuts in the options page.
  • Change (Mac OS): Keyboard shortcut Opt + Del has been reverted back to Ctrl + Del on Mac computers only.

The next update will bring further improvements to keyboard shortcut functionality and will provide more customisation.

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or want to leave feedback, please submit a ticket in the bug tracker or tweet me at @TypioRecovery.
