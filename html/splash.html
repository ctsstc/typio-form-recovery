v1.6.5 Updated!

This update improves site support, addresses some usability issues and fixes keyboard shortcuts hopefully for the last time. I've added the ability to set your own keyboard combinations and I've changed the defaults to not cause issues on any systems. See detailed keyboard shortcut changes below or see the options page to view or change the defaults.

Keyboard shortcut related improvements:

  • New: You can now set your own keyboard shortcuts in the options page (more options coming).
  • Change (Mac OS): Keyboard shortcut Opt + Delete has been changed to Ctrl + Backspace.
  • Change (Windows OS): Keyboard shortcut Alt + Delete has been changed to Alt + Backspace.

Other general improvements:

  • New: Added support for websites with XML content types (battle.net).
  • New: Added edge detection for positioning right-click popup menu.
  • New: Right-click popup menu no longer hides current session data.

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or want to leave feedback, please submit a ticket in the bug tracker or tweet me at @TypioRecovery.
