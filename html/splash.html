v1.4.0 Updated!

This update has been in the works for quite a while and brings lots of exciting changes! Apart from the new features introduced, it will also make future development a lot easier as the code base has been refactored and cleaned up. Here's what's new in this version!

  • New: Save indicator to reassure you your text is being saved as you type. Has various styles to choose from and can be disabled in settings.
  • New: Support for chat apps (dynamically cleared inputs)
  • New: Keyboard shortcuts! Press Ctrl + Delete once to open recovery dialog, press twice to restore previous session.
  • New: Context menu now also shows recent entries from other fields
  • New: Added option for showing/hiding small values in recovery dialog
  • New: Support for nested iframes/shadow dom

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or would like to leave feedback, please submit a ticket in the bug tracker.
