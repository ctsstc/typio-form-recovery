2.3.0 Updated!

New things in this version

This update brings some major arcitectural changes to the code which will improve support further. bl bla spa support iframes shadowdom

This update brings major improvements to the modularity of the code and should make it a lot easier to get a grasp on if you want to contribute.

  • New: Save indicator to reassure you your text is being saved as you type. Has various styles to choose from and can be disabled in settings.
  • New: Added option for showing/hiding small values in recovery dialog
  • New: Displays input type in recovery dialog
  • New: Keyboard shortcuts! Press Ctrl + Delete once to open recovery dialog, press twice to restore previous session.
  • Change: "Pick recovery target" replaced with Copy button.

For nerds:

  • Fix: All injected html is now within one parent shadowdom (used multiple before but that's deprecated).
  • Fix: Improved error message if trigger context before page load etc
  • Fix: Support for nested shadow dom and iframes
  • Fix: Concatinated all shadowroot css
  • Fix: Support for inputs that are dynamically cleared (support for chat apps like fb messenger)
  • Fix: Better performance in context menu when big entries are hovered (added truncation)

If you encounter any bugs or want to leave feedback, please create a ticket in the bug tracker.
