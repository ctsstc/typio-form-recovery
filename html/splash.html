v1.5.0 Updated!

This update is the result of a battle with some pesky bugs I introduced in the last version. Also, some new cool features I think you'll enjoy.

  • New: Better display of field type in recovery dialog
  • New: Redesigned options page
  • Fix: Various animation improvements (for save indicator and recovery dialog)
  • Critical fix: Critical bugfix in frame injection script (see report 1, report 2)

If you want the nitty gritty details, head over to the repository page where you can see the full changelog with technical details. If you encounter any bugs or would like to leave feedback, please submit a ticket in the bug tracker.
